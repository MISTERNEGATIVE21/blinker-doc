# 实时数据传输  
自blinker App 2.5.9开始，开始提供数值型数据实时传输方法。
为降低公共服务器资源消耗，请务必遵照本文档进行实时数据传输。

## 通信指令  
1. 开启实时数据后，app每9秒发起一次实时数据获取  
```json
{"realtime":["humi","temp"]}
```
2. 设备反馈10次实时数据  
```json
{
    "humi":{
        "val":100,
        "date":1605373409,
    },
    "temp":{
        "val":100,
        "date":1605373409,
    }
}
```  
> date单位为秒  
  
## 组件显示  
**数据组件** 在组件编辑页面中，勾选`启用实时数据`，即会实时更新数据。
**图表组件** 切换到实时模式，组件在收到指令数据后，如key和存储key相同，即会绘制出相应的数据的实时曲线。

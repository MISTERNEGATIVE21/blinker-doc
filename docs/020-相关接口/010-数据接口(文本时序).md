# 文本时序数据接口beta  

> 文本时序数据相当于日志记录，**该功能仅限专业版使用**



## 存储接口

### 存储方式

- 请求方式：`POST /api/v1/device/storage/logs`

- 请求参数：

  ```json
  {
      "device": "",	// 设备名
      "token": "",    // 设备token (设备上线返回值中的iotToken)
      "data": [["timestamp", "log1"]]
  }
  ```

- 返回参数：

  ```json
  {
      "message": 1000,
      "detail": ""
  }
  ```
